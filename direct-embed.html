PFLX Class Tools - Widget Demo

This is a working embed of the PFLX Class Tools widget. The code below can be used to embed this widget on any website.

Embed Code

Copy and paste this code to embed the widget on your website:

<!-- PFLX Class Tools Widget (16:6 Aspect Ratio) -->
<style>
  .pflx-embed-container {
    position: relative;
    width: 100%;
    padding-bottom: 37.5%; /* 6/16 = 0.375 or 37.5% */
    height: 0;
    overflow: hidden;
    border-radius: 8px;
    margin: 20px 0;
    box-shadow: 0 4px 10px rgba(0, 0, 0, 0.1);
  }
  
  .pflx-embed-container iframe {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    border: 0;
  }
</style>

<div class="pflx-embed-container">
  <iframe 
    src="https://pflx-class-tools.vercel.app/?embed=true" 
    allowfullscreen></iframe>
</div>

Usage Notes

  • The widget maintains a 16:6 aspect ratio on all screen sizes
  • You can adjust the width of the container by changing the width property (currently set to 100%)
  • The embed code works on any HTML page or CMS that supports custom HTML
  • For security reasons, some platforms may require you to add the domain to their CSP policy
