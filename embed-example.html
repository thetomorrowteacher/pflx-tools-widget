PFLX Class Tools - Embed Example

This page demonstrates how to embed the PFLX Class Tools widget with a 16:6 aspect ratio. The widget below is a live example.

HTML Embed Code:
<div class="embed-container">
  <iframe 
    src="https://pflx-class-tools.vercel.app/?embed=true" 
    frameborder="0" 
    allowfullscreen>
  </iframe>
</div>

<style>
  .embed-container {
    width: 100%;
    position: relative;
    padding-bottom: 37.5%; /* 16:6 aspect ratio */
    height: 0;
    overflow: hidden;
  }
  .embed-container iframe {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    border: 0;
  }
</style>

To customize the widget width, change the width of the .embed-container element. The aspect ratio will remain 16:6.
